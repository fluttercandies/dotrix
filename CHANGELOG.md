## 1.0.0

- Initial release of Dotrix
- 28 built-in dot matrix animation indicators
  - Basic: Pulse, Spinner, Wave
  - Extended: ArrowMove, Burst, Chaos, Clockwise, Corners, Cross, Diamond, Pulse2, SineWave
  - Advanced: Spiral, Zigzag, Radial, Cascade, Inward, Ripple, Checkerboard, Converge, Scatter, Flip, PulseRing, DiagonalRace
  - Flow: FlowUp, FlowDown, FlowLeft, FlowRight
- Fully customizable dot size, colors, spacing, glow effects, and animation speed
- Zero external dependencies
- Auto-playing infinite loop animations
- Production ready with comprehensive documentation and examples
